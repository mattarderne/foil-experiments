DW Foil Experiment Browser

Browse the foil experiments without digging through code.

Pick a tab, open the main demo, and swipe through the media like a short story. The site is meant to be the simple public front door to the structural, simulation, and motion experiments in the repo.

Non-Technical Setup

Use Claude Cowork if you want help cloning, installing, and running the repo.

If you do not want to work in a terminal yourself, a practical route is Claude Cowork in Claude Desktop. Based on the way this site is meant to be used, you can ask Cowork to clone the repo, install dependencies, and run the viewers for you. Start with the official Cowork getting-started guide.

1. Start with Claude Pro Pro is the best-cost starting point for most individuals. It is the cheapest paid tier, and Cowork is available on paid Claude plans.
2. Ask Cowork to do the setup Tell it to clone dwfoil-com/foil-experiments, install what the repo needs, and run the website or viewers you want to explore.
3. Move to Max 5x only if needed If you expect long or frequent Cowork sessions, Max 5x is the next sensible step up. It gives much more session capacity, but costs substantially more.

Once you know what you actually want to optimize, the more advanced workflow is a long-running Karpathy loop: ask the agent to keep iterating on one narrow objective, review results, adjust the target, and continue. Cowork is a good way to start; a tighter experiment loop is better once the objective is clear.
