**Status:** poc

This project defines a controlled experiment to compare a Tuttle box against a regular dual-track mast cassette in the same TT60 foil board. The comparison isolates insert-to-board load transfer using the same outer shell, the same stance geometry, and the same mast-side force and moment resultants.

The key design choice is to model the shell, EPS core, and HD foam insert as separate material regions with different bond paths. Tuttle is bonded top and bottom; tracks are bonded bottom only. The included viewer prototype makes the intended side-by-side 3D comparison concrete before the full FEA result pipeline is implemented.
